What is ORBFORGE?

Your agent should have a face. ORBFORGE forges a living animated orb — a burning ring, a glassy globe, a plasma core, a wireframe sculpture — and hands you a looping file you can drop into a bot, an app, or a profile in seconds.

Shape it

  • Drag a slider or type an exact value. It renders live, and a pink marquee traces the exact part of the orb you’re changing.
  • Randomize rolls a whole new species — ring, plasma, glass, mesh, sculpture, aura. Each roll gets a seed that rebuilds it exactly; the seed rides along in every save, export, and downloaded file.
  • The preset manager loads, renames, and stacks presets — tick up to three to layer them into one composite orb.

Take it with you

  • WebP — small, smooth, true soft transparency. GIF — plays anywhere.
  • Every downloaded file carries its full recipe inside it. Drop it back into Import to keep editing — no separate config to keep track of.
  • Set a target size and the exporter fits it for you.

It never leaves your browser

No login, no upload, no tracking. Everything renders and encodes on your machine — the page’s own security policy forbids it from phoning home.
